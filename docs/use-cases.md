# Use Cases

## Actors
- **Donor**: Register, donate, view history
- **Hospital Staff**: Request blood, manage inventory
- **Blood Bank Manager**: Manage inventory, approve requests
- **Admin**: Manage users, system settings

## UC1: Register Donor
Fill form → Verify email → Account created

## UC2: Schedule Appointment
Select center → Pick date/time → Get confirmation

## UC3: Request Blood
Enter details → Submit → Track status

## UC4: Manage Inventory
View stock → Allocate → Track expirations

## UC5: Process Donation
Record donation → Update inventory → Update history

## UC6: View History
Login → View → Filter/Search

## UC7: Urgent Requests
Mark urgent → Priority processing → Notify donors

## UC8: User Management
Create/Edit/Delete users → Assign roles

## UC9: Reports
Select type → Generate → Export

## UC10: Update Profile
Edit info → Save → Confirmation

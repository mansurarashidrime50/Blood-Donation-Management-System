# System Design

## Architecture
Client (Web/Mobile) → API Gateway → Microservices → Database

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js |
| Mobile | React Native |
| Backend | Node.js + Express |
| Language | TypeScript |
| Database | PostgreSQL |
| Cache | Redis |
| Auth | JWT |
| Container | Docker |
| Orchestration | Kubernetes |
| Cloud | AWS |

## Microservices
- User Service
- Donation Service
- Inventory Service
- Request Service
- Notification Service
- Report Service

## Features
- RESTful APIs
- JWT Authentication
- RBAC
- Real-time notifications
- Auto-scaling
- 99.9% uptime

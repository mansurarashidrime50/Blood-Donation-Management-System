# Functional Requirements
## User Management
- Register users (donors/hospitals/admins)
- Login/Logout with email & password
- Password reset
- Update profile
- Role-based access

## Donor Management
- Register with personal details
- Track blood type & medical history
- Check eligibility
- View donation history
- Schedule appointments

## Blood Inventory
- Track all blood units (type/quantity)
- Auto-update on donation
- Track expiration dates
- Allocate to hospitals
- Quarantine management

## Blood Requests
- Hospitals request blood
- Emergency priority
- Track status (pending/approved/fulfilled)
- Notify donors for urgent needs
- Blood type matching

## Notifications
- Appointment reminders
- Blood usage alerts
- Urgent need alerts
- Low stock alerts
- Eligibility updates

## Reports
- Donation statistics
- Inventory status
- Donor history
- Blood usage
- Analytics dashboard

## Appointments
- Book appointments
- View available slots
- Send confirmations
- Reschedule
- Track attendance

# Interviews

---

## Interview 1: Hospital Admin

| Question | Answer |
|----------|--------|
| Biggest challenge? | "We run out of blood in emergencies" |
| How track inventory? | "Manual records and spreadsheets" |
| What would help? | "Real-time tracking and alerts" |
| Concerns? | "Data security" |

**What We Learned:**
- Manual tracking is bad
- Need real-time system
- Security matters

## Interview 2: Donor

| Question | Answer |
|----------|--------|
| Why donate? | "To save lives" |
| What frustrates you? | "Process takes too long" |
| What would help? | "Online booking and reminders" |
| How contact? | "SMS or WhatsApp" |

**What We Learned:**
- Process is too slow
- Need booking system
- SMS is best

## Interview 3: Recipient

| Question | Answer |
|----------|--------|
| Experience? | "Stressful and time-consuming" |
| What would help? | "Real-time availability" |

## Key Findings

| Problem | Solution Needed |
|---------|-----------------|
| Manual tracking | Digital system |
| No alerts | SMS alerts |
| Long process | Easy registration |
| No booking | Online scheduling |
| Limited hours | 24/7 access |
